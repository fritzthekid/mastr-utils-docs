Introduction

mastr-utils is a collection of Python tools and a webapp for analyzing and visualizing GPS tracks and related geospatial data. It is designed for scientific, educational, and personal use, supporting both command-line and browser-based workflows.

The documentation covers installation, usage, examples, troubleshooting, and more.

Features

  • Interactive webapp for uploading, viewing, and analyzing GPS data
  • Command-line tools for batch processing and scripting
  • Plot generation: elevation profiles, speed analysis, map overlays, etc.
  • GPX file support and conversion
  • Extensible modular design

Installation

  1. Clone the repository from GitHub: 
    git clone https://github.com/fritzthekid/mastr-utils-docs
  2. Install dependencies: 
    pip install -r requirements.txt
  3. Start the webapp: 
    python -m mastr_utils.webapp

Usage

Webapp

Launch the web application and open your browser at http://localhost:5000. Upload GPX files and explore interactive plots and statistics.

Command-line Tools

Example: Generate an elevation plot from a GPX track: 

python -m mastr_utils.tools.plot_elevation track.gpx -o elevation.png

Plot Types

  • Elevation profiles
  • Speed analysis
  • Map overlays

Examples

See the Examples chapter for detailed usage and output samples.

Troubleshooting

  • Cannot start webapp: Check Python version & dependencies.
  • Missing modules: Run pip install -r requirements.txt again.
  • Plot not rendering: Verify input GPX file format.
